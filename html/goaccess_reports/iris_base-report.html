Uno strumento molto utile per analizzare i file di log del web-server in tempo reale, oltretutto disponibile nei repository di CentOS.
GoAccess

Per installarlo:
yum install goaccess.x86_64

** per qualunque versione bisogna lanciare goaccess con i privilegi di root per poter leggere il file access_log **

Versione 1.2-1

Per la creazione del file statico lanciare il comando:
goaccess -f /var/log/httpd/access_log -a > /var/www/html/report.html

Un'analisi più accurata pare sia possibile con il comando:
goaccess /var/log/httpd/access_log -o /var/www/html/report.html --log-format=COMBINED

Una nuova funzione real-time è stata introdotta, lanciando il comando:
goaccess /var/log/httpd/access_log -o /var/www/html/report.html --log-format=COMBINED --real-time-html

Per filtrare i report in base a delle pagine specifiche personalizzando il titolo della pagina:
zgrep -h -i 'TYPE=iris_base' /var/log/httpd/access_log* | goaccess --html-report-title='IRIS BASE' -a -o /var/www/html/goaccess_reports/iris_base-report.html --log-format=COMBINED
